# Описание проекта

## Бизнес-контекст
Компания является маркетплейсом товаров с рекламными блоками в мобильном приложении.

## Проблема
Текущее расположение рекламного блока (после 3 линий товаров) может быть не оптимальным для вовлечения пользователей.

## Гипотеза
Размещение рекламного блока над основной секцией товаров увеличит взаимодействие пользователей с рекламой.

## Метрики
- Основная: CTR (кликабельность)
- Вспомогательные: CR, ARPU, ARPPU
