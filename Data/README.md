# Описание данных

## data_raw.csv
Исторические данные о действиях пользователей до эксперимента.

Столбцы:
- `dt`: Дата события
- `client_id`: Идентификатор пользователя
- `event`: Тип события (view/click/add)

## final_results_to_analyze.xlsx
Результаты А/Б эксперимента.

Столбцы:
- `date`: Дата
- `client_id`: Идентификатор пользователя
- `ab_group`: Группа эксперимента (control/test)
- `is_view_ads`: Флаг просмотра рекламы
- `cnt_view_ads`: Количество просмотров рекламы
- `is_adds_ads`: Флаг добавления товаров
- `cnt_adds_ads`: Количество добавлений
- `sum_adds_ads`: Сумма добавленных товаров (руб)
- `is_orders_ads`: Флаг заказов
- `cnt_orders_ads`: Количество заказов
- `sum_orders_ads`: Сумма заказов (руб)
