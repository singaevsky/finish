### –î–∏—Å–ø–µ—Ä—Å–∏—è ratio-–º–µ—Ç—Ä–∏–∫–∏ (–¥–µ–ª—å—Ç–∞-–º–µ—Ç–æ–¥)

```python
def var_ratio(x, y):
    mean_x = np.mean(x)
    mean_y = np.mean(y)
    var_x = np.var(x, ddof=1)
    var_y = np.var(y, ddof=1)
    cov_xy = np.cov(x, y, ddof=1)[0, 1]
    return var_x/mean_y**2 + var_y*mean_x**2/mean_y**4 - 2*mean_x*cov_xy/mean_y**3

var_cr = var_ratio(adds, clicks)
std_cr = np.sqrt(var_cr)
```

### –†–∞—Å—á—ë—Ç MDE

–ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º:
- –û–±—â–µ–µ —á–∏—Å–ª–æ –∞–∫—Ç–∏–≤–Ω—ã—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π –∑–∞ –Ω–µ–¥–µ–ª—é ‚Äî **N = 100 000** (–Ω–∞ –æ—Å–Ω–æ–≤–µ `data_raw.csv`);
- –ú—ã –º–æ–∂–µ–º –∑–∞–¥–µ–π—Å—Ç–≤–æ–≤–∞—Ç—å **–Ω–µ –±–æ–ª–µ–µ 20%** ‚Üí **–º–∞–∫—Å–∏–º—É–º 20 000 –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π** (10 000 –≤ –∫–æ–Ω—Ç—Ä–æ–ª—å, 10 000 –≤ —Ç–µ—Å—Ç);
- –£—Ä–æ–≤–µ–Ω—å –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏: **Œ± = 0.05**, –º–æ—â–Ω–æ—Å—Ç—å **1‚àíŒ≤ = 0.8**.

```python
def get_MDE(mu, std, sample_size, n_groups=2, alpha=0.05, beta=0.2):
    t_alpha = stats.norm.ppf(1 - alpha/2)
    t_beta = stats.norm.ppf(1 - beta)
    # –î–ª—è —Ä–∞–≤–Ω—ã—Ö –≥—Ä—É–ø–ø: r = 1, target_share = 0.5
    mde_abs = (t_alpha + t_beta) * std * np.sqrt(2 / (sample_size / n_groups))
    mde_pct = mde_abs / mu * 100
    return mde_abs, mde_pct

sample_size = 20000  # 10k + 10k
mde_abs, mde_pct = get_MDE(mu, std_cr, sample_size)
```

## 4. –ê–Ω–∞–ª–∏–∑ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–≤ —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç–∞

### –†–∞—Å—á—ë—Ç –∫–ª—é—á–µ–≤–æ–π –º–µ—Ç—Ä–∏–∫–∏ –ø–æ –≥—Ä—É–ø–ø–∞–º

def calc_cr(group):
    total_clicks = group['cnt_view_ads'].sum()  # –≤ —Ñ–∞–π–ª–µ: cnt_view_ads = –∫–ª–∏–∫–∏?

> üìå **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ**: –≤ –¥–∞–Ω–Ω—ã—Ö `cnt_view_ads` –Ω–∞ —Å–∞–º–æ–º –¥–µ–ª–µ ‚Äî **–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–ª–∏–∫–æ–≤**, –Ω–µ –ø—Ä–æ—Å–º–æ—Ç—Ä–æ–≤ (—Å—É–¥—è –ø–æ –∫–æ–Ω—Ç–µ–∫—Å—Ç—É: `is_view_ads` ‚Äî —Ñ–ª–∞–≥ –∫–ª–∏–∫–∞). –≠—Ç–æ –≤–∞–∂–Ω–æ!
> –£—Ç–æ—á–Ω–µ–Ω–∏–µ:
> - `is_view_ads` = –±—ã–ª –∫–ª–∏–∫ –ø–æ —Ä–µ–∫–ª–∞–º–µ ‚Üí –ª—É—á—à–µ –Ω–∞–∑—ã–≤–∞—Ç—å `is_click_ads`
> - `cnt_view_ads` = —á–∏—Å–ª–æ –∫–ª–∏–∫–æ–≤

–ü–æ—ç—Ç–æ–º—É:
- **CTR** = `is_click_ads / –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π` (–Ω–æ —É –Ω–∞—Å –Ω–µ—Ç –æ–±—â–µ–≥–æ —á–∏—Å–ª–∞ –ø—Ä–æ—Å–º–æ—Ç—Ä–æ–≤ –≥–ª–∞–≤–Ω–æ–π ‚Üí CTR —Å—á–∏—Ç–∞—Ç—å –Ω–µ–ª—å–∑—è –±–µ–∑ `views`)
- **CR** = `cnt_adds_ads / cnt_view_ads` ‚Äî –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ.

### –°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∞—è –∑–Ω–∞—á–∏–º–æ—Å—Ç—å (t-—Ç–µ—Å—Ç –¥–ª—è ratio-–º–µ—Ç—Ä–∏–∫)

–ò—Å–ø–æ–ª—å–∑—É–µ–º **Welch‚Äôs t-test** –∏–ª–∏ **bootstrap**. –ü—Ä–æ—â–µ ‚Äî **bootstrap**:

```python
def bootstrap_metric(group, n_boot=10000):
    adds = group['cnt_adds_ads'].values
    clicks = group['cnt_view_ads'].values
    crs = []
    for _ in range(n_boot):
        idx = np.random.choice(len(group), len(group), replace=True)
        a = adds[idx].sum()
        c = clicks[idx].sum()
        crs.append(a / c if c > 0 else 0)
    return np.array(crs)

boot_test = bootstrap_metric(test)
boot_control = bootstrap_metric(control)

p_value = (boot_test <= boot_control.mean()).mean()
# –∏–ª–∏ –¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω–∏–π:
p_value = 2 * min(
    (boot_test <= boot_control.mean()).mean(),
    (boot_test >= boot_control.mean()).mean()
)
```
